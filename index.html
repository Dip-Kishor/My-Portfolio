Wheat Yield Prediction System

The Wheat Yield Prediction System is an innovative application designed to predict wheat production using a blend of advanced machine learning techniques and a robust software architecture. The system integrates a Python-based machine learning model, a .NET-based backend, and a ReactJS frontend to deliver accurate and accessible predictions to its users.

System Architecture and Contributions:


1. Machine Learning (Python):

The core of the system is a machine learning model developed using Python. The model predicts wheat yield based on key agricultural parameters, including rainfall, temperature, soil composition, and more. Development was led by Surakshya Adhikari, with significant contributions from Dip Kishor Regmi.


2. Backend (.NET):

The backend, built in .NET, provides a secure and efficient API for communication between the frontend and the ML model. It includes features such as user authentication, data management, and ML integration. Developed by Dip Kishor Regmi.


3. Frontend (ReactJS):

The frontend is a user-friendly interface created using ReactJS. It allows users to input data, view predictions, and interact seamlessly with the system. Designed and implemented by Prajwal Baral.



This collaborative effort combines expertise in software development, machine learning, and web design to create a powerful tool for farmers, researchers, and agricultural planners.
